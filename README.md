# 🎮 Como Funcionaria o “Escape Room Inteligente”

A ideia é criar um mini jogo web onde o usuário precisa resolver enigmas para “escapar” de um sistema bloqueado.

O segredo é:

> parecer cinematográfico e interativo.

Vocês não precisam fazer algo gigantesco.
Mesmo com 3–5 fases já fica MUITO forte.

---

# 🧠 História do Projeto (importante para impressionar)

## Exemplo de narrativa:

> “Uma inteligência artificial perdeu o controle do laboratório digital e bloqueou o sistema. O jogador tem 5 minutos para resolver os enigmas e recuperar o acesso.”

Isso já deixa a apresentação MUITO mais interessante.
